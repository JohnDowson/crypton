# Crypton
### a note taking application


## Installation

```sh
git clone git@github.com:JohnDowson/crypton.git
cargo install --path ./crypton
```

## Usage

```sh
crypton <SUBCOMMAND>
    edit    <hash>
    list
    new     [contents]
    remove  <hash>
```
DB location can be configured using `CRYPTON_DB` enviroment variable.
